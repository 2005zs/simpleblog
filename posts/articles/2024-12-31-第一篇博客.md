## 新建博客
```bash
npm run new
```
## 预览博客

```bash
npm run preview
``` 
## priority表示博客的优先级，数字越大，优先级越高
```bash
0 普通
1 普通
>1 置顶
```

## 推送github
```bash
git push -u https://name@github.com/name/repo.git master
```
## 使用ssh
```bash
git remote remove origin
git remote add origin git@github.com:name/repo.git
git push -u origin master
```
